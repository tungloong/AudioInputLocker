# Code of Conduct

AudioInputLocker is a small open-source macOS app. Please keep discussion
practical, respectful, and focused on improving the project.

## Expected Behavior

- Be kind and direct.
- Assume good intent when reviewing issues and pull requests.
- Keep feedback about the work, not the person.
- Avoid harassment, insults, discrimination, or intentionally disruptive
  behavior.

## Reporting

For normal project issues, use GitHub Issues.

For security-sensitive reports, follow `SECURITY.md` and do not post exploit
details publicly.
