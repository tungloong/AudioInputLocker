# Privacy Policy

Effective date: 2026-05-23

AudioInputLocker is a local macOS menu bar utility for managing the system
default audio input device.

## Data Collection

AudioInputLocker does not collect, store, transmit, sell, or share personal
data.

The app does not include analytics, advertising, tracking, accounts, telemetry,
or third-party SDKs.

## Local Settings

AudioInputLocker stores the user's preferred input device and lock setting
locally on the Mac using `UserDefaults`. This information is not transmitted
off the device.

## Network Access

AudioInputLocker does not make network requests.
