# AudioInputLocker

AudioInputLocker is a small macOS menu bar app for managing and locking the
system default audio input device.

- [GitHub repository](https://github.com/tungloong/AudioInputLocker)
- [Preview release](https://github.com/tungloong/AudioInputLocker/releases/tag/v0.1.0-preview)
- [All releases](https://github.com/tungloong/AudioInputLocker/releases)
- [Privacy policy](privacy.html)
- [Support](https://github.com/tungloong/AudioInputLocker/issues)

AudioInputLocker works locally on your Mac and does not collect data.
