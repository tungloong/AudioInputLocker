# Support

AudioInputLocker is a small open-source macOS utility. Support is best effort.

## Questions And Bugs

Use GitHub Issues for:

- Reproducible bugs.
- Feature requests.
- Documentation gaps.
- Compatibility reports for specific microphones, headsets, or macOS versions.

Before opening an issue, please include:

- macOS version and Mac model.
- Xcode version if the problem is build-related.
- Audio device name and connection type when relevant.
- Steps to reproduce.
- Whether the app was built from source or installed from a release build.

## Security Reports

Do not include security-sensitive details in a public issue. See `SECURITY.md`
for the reporting path.
